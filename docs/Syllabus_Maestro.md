# PROJECT STRATEGY: CONTENT MERGER

## 1. Core Content Source
Use "Aprendamos Español I, II, III" (the PDF from Cuba) as the 
primary source for vocabulary, grammar rules, and reading texts.

## 2. Logical Structure Source
Use "Aula América 1" as the visual and pedagogical model. 
Every lesson must be divided into:
- **Understand:** Introduction with real-life context
- **Explore:** Grammar and vocabulary discovery
- **Practice:** Exercises to fix knowledge
- **Communicate:** A final task (Action)

## 3. Personalization Engine
The platform adapts to each student's:
- Native language (L1) — not assumed to be Dutch
- Personal interests and hobbies
- Learning goals (exam prep vs free learning)
- Available time per day

## 4. Two Learning Tracks
- **Dutch Track:** Full SLO 2026 compliance for Havo/Vwo students.
  Maps all units to Kerndoelen 37, 38, 39 at ERK A1/A2 levels.
- **Open Track:** Free, interest-based Spanish learning for anyone.
  No exam pressure. Same quality content.

## 5. Legal Compliance
Dutch Track units must map to SLO 2026 Kerndoelen for 
Moderne Vreemde Talen (Kerndoelen 37, 38, 39).

## 6. Language Base
- Default: Dutch-Spanish
- Alternative: English-Spanish
- Future: Any L1-Spanish (Arabic, Turkish, etc.)
- Language preference stored in user profile

## 7. Cultural Scope
Content covers ALL Spanish-speaking cultures:
- Caribbean (Cuba, Puerto Rico, Dominican Republic)
- South America (Argentina, Colombia, Peru, Chile)
- Central America (Mexico, Costa Rica, Nicaragua)
- Spain (Madrid, Barcelona)
